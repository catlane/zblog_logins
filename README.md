zblog_logins
======

zblog登录模块（最全的第三方登录）

一个针对于zblog开发的登录插件，集成了多个主题，集成了多种第三方登录方式（qq、微博、github），让您一个插件满足您所有需求

## 安装

```
$ composer require catlane/zblog_logins
然后将文件夹放到 /zb_users/plugin/中
```
